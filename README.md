# 🕊️ Conversando com Deus — Assistente Espiritual com IA

**Conversando com Deus** é uma plataforma moderna e acolhedora de apoio espiritual, desenvolvida para oferecer orientação bíblica, devocionais personalizados e estudos bíblicos através de Inteligência Artificial de ponta (Google Gemini).

![Design Preview](https://github.com/user-attachments/assets/placeholder)<!-- Adicione um screenshot aqui após o deploy -->

## ✨ Funcionalidades principais

- **Apoio Espiritual 24h:** Converse sobre suas dúvidas, angústias e vitórias sob uma perspectiva cristã.
- **Modelos de Estudo Bíblico:** Gere roteiros para células e pequenos grupos em segundos.
- **Devocionais Personalizados:** Crie meditações diárias baseadas no seu estado emocional ou tema de interesse.
- **Preparação de Pregações:** Auxílio na estruturação de esboços e busca por referências bíblicas.
- **Interface Premium:** Design "Google Antigravity" com tema claro e escuro, focado na paz e na experiência do usuário.

## 🚀 Tecnologias Utilizadas

- **Frontend:** [React.js](https://reactjs.org/) + [Vite](https://vitejs.dev/)
- **Estilização:** CSS Vanilla com Design System Premium
- **Inteligência Artificial:** [Google Gemini API](https://aistudio.google.com/)
- **Ícones:** [Lucide-React](https://lucide.dev/)
- **Animações:** CSS Transitions & Animations

## 🛠️ Como rodar o projeto localmente

1. **Clone o repositório:**
   ```bash
   git clone https://github.com/seu-usuario/conversando-com-deus.git
   cd conversando-com-deus
   ```

2. **Instale as dependências:**
   ```bash
   npm install
   ```

3. **Configure as variáveis de ambiente:**
   - Renomeie o arquivo `.env.example` para `.env`.
   - Insira sua `VITE_GEMINI_API_KEY`.

4. **Inicie o servidor de desenvolvimento:**
   ```bash
   npm run dev
   ```

## 📜 Isenção de Responsabilidade
Esta é uma ferramenta de apoio baseada em Inteligência Artificial. Ela **não substitui** o aconselhamento de um pastor, líder espiritual ou profissional de saúde mental. Em casos de crise ou urgência, procure ajuda humana imediata.
